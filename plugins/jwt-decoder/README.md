# JWT Decoder Plugin

Decode and inspect JSON Web Tokens (JWT).

## Features

- **Decode** — split token into header, payload, and signature
- **Timestamp parsing** — human-readable `iat`, `exp`, `nbf`
- **Expiry detection** — highlights expired tokens
- **No external dependencies** — pure client-side decoding

## Usage

Paste a JWT token and click **Decode** to inspect its contents.
