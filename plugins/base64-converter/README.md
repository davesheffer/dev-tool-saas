# Base64 Converter

Encode and decode Base64 strings easily.

## Features

- Real-time encoding and decoding
- Syntax validation
- Clean and simple UI
- Supports large text inputs

## Usage

1. Select **Encode** or **Decode** mode at the top right.
2. Type or paste your text into the **Input** box.
3. The result will strictly appear in the **Output** box.
