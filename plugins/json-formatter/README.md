# JSON Formatter Plugin

Format, validate, and minify JSON with a clean UI.

## Features

- **Pretty print** — formats JSON with 2-space indentation
- **Minify** — compresses JSON to a single line
- **Validation** — clear error messages for malformed JSON
- **Copy** — one-click copy to clipboard

## Usage

Paste any JSON into the input area, then click **Format** or **Minify**.
